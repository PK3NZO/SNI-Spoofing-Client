# Security Policy

## Reporting a Vulnerability

If you believe you found a security issue:

1. Do not publish exploit details in a public issue.
2. Use GitHub Security Advisories if enabled for this repository.
3. If advisories are not enabled yet, open a minimal issue without disclosing the exploit path and ask for a private contact channel.

Please include:

- affected version
- macOS architecture (`arm64` or `x86_64`)
- whether the issue affects `Proxy`, `Tunnel`, or both
- reproduction steps
- expected impact
